# Lưu trữ trên Cloudflare R2 (khuyên dùng)

R2 = object storage kiểu S3, **miễn phí egress** (không tính phí băng thông tải về).
Bonus lớn: đẩy file + manifest lên R2 → **iOS cài OTA được mà KHỎI cần domain hay
self-signed cert** cho server (R2 đã có HTTPS valid của Cloudflare).

App tự bật R2 khi có đủ 5 biến env; thiếu → tự quay về lưu disk. Không phải sửa code.

---

## 1. Tạo bucket

1. Cloudflare Dashboard → **R2** → **Create bucket**.
2. Đặt tên, ví dụ `appdrop`. Region: Automatic. Create.

## 2. Bật Public URL (để tải/cài không cần đăng nhập)

Trong bucket → tab **Settings** → **Public access**:
- Cách nhanh: bật **R2.dev subdomain** → được URL kiểu `https://pub-xxxxxxxx.r2.dev`.
- Hoặc gắn **Custom Domain** (vd `cdn.congty.com`) nếu muốn URL đẹp (cần domain + DNS trên Cloudflare).

→ URL này là **`R2_PUBLIC_BASE_URL`**.

## 3. Tạo API token (Access Key)

R2 → **Manage R2 API Tokens** → **Create API token**:
- Permission: **Object Read & Write**.
- Scope: bucket vừa tạo (hoặc all buckets).
- Create → copy ngay 2 giá trị (chỉ hiện 1 lần):
  - **Access Key ID** → `R2_ACCESS_KEY_ID`
  - **Secret Access Key** → `R2_SECRET_ACCESS_KEY`

## 4. Lấy Account ID

Trang R2 Overview (góc phải) hoặc URL dashboard có **Account ID** → `R2_ACCOUNT_ID`.

## 5. Điền vào `.env`

```
R2_ACCOUNT_ID=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
R2_ACCESS_KEY_ID=xxxxxxxxxxxxxxxxxxxx
R2_SECRET_ACCESS_KEY=xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
R2_BUCKET=appdrop
R2_PUBLIC_BASE_URL=https://pub-xxxxxxxx.r2.dev
```

Restart app → upload thử. File sẽ nằm trên R2 (`builds/<slug>/build.apk`), không còn trong máy.

---

## Cách hoạt động sau khi bật

```
Upload → app stage tạm → parse metadata → đẩy build/icon(/manifest iOS) lên R2 → xoá bản tạm
Tải/Cài → redirect thẳng R2 public URL (không tốn băng thông server)
iOS     → itms-services trỏ manifest.plist trên R2 (HTTPS Cloudflare) → cài luôn, khỏi domain
Hết hạn → job tự xoá object trên R2
Metadata (tên/version) vẫn ở SQLite local (nhẹ) — chỉ file lớn nằm R2
```

## Lưu ý

- **Password từng link**: file trên R2 public = ai có URL đều tải được → mật khẩu link
  chỉ ẩn ở trang, không chặn tuyệt đối. Mật khẩu login upload (`APP_PASSWORD`) vẫn nguyên.
- Không cần cấu hình CORS (chỉ GET file public).
- Bảo mật: `.env` chứa Secret Access Key → **không commit** (đã gitignore).
